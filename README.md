# Web Scraper

Pulls data from https://food.oregonstate.edu & https://my.uhds.oregonstate.edu/api/dining/calendar to get dining center hours and menu items

# Technologies used

- Node.js v18+
- Puppeteer
- Jest

# Install

```cmd
npm i
```

# Run

```cmd
node index.js
```

# Test

```cmd
npm test test.js
```
